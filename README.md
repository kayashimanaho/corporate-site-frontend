# コーポレートサイト フロントエンド (Next.js)

このプロジェクトは、Next.js を使用して構築されたコーポレートサイトのフロントエンドです。WordPress REST API からデータを取得してお知らせ一覧や詳細ページを表示します。

## 開発の始め方

まず、開発サーバーを起動します：

```bash
npm run dev


## コミットメッセージの規約

このプロジェクトでは、以下の接頭辞を使用してコミットメッセージを記述します。

| 接頭辞 | 意味 |
| :--- | :--- |
| **feat:** | 新しい機能の追加 |
| **fix:** | バグの修正 |
| **docs:** | ドキュメント（READMEなど）のみの変更 |
| **style:** | 空白、フォーマット、セミコロン追加など（動作に影響なし） |
| **refactor:** | 仕様に影響がないコード改善（リファクタリング） |
| **perf:** | パフォーマンス向上に関連する変更 |
| **test:** | テストコードの追加や修正 |
| **chore:** | ビルド、補助ツール、ライブラリの管理関連（npm installなど） |
